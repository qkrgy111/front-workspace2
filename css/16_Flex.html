# display: flex

  • 해당 요소의 display 속성을 flex로 변경하면 flex-box가 된다
  • 해당 박스 내부에 있는 자식 요소들은 자동적으로 flex-item이 된다
  • flex-box는 내부 자식 요소들의 크기를 유연하게 조절해준다

# order property (flex-item으로 설정된 요소에 적용되는 속성)

  • flex-item들의 order 속성의 기본값은 0이다
  • 자기보다 속성값이 크면 뒤에 나오고 작으면 앞에 나온다
