# clear 속성 사용하기

  • clear: left - 해당 요소의 왼쪽에 float 요소를 올 수 없게 만든다
  • clear: right - 해당 요소의 오른쪽에 float 요소를 올 수 없게 만든다
  • clea: both - 해당 요소 양쪽에 float 요소를 올 수 없게 만든다
