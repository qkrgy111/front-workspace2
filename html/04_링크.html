# 다른 웹 페이지로 이동할 수 있게 링크 걸기

- a태그 사용해 링크 걸기
- 자바스크립트로 ㅇ링크 걸기

# a태그 사용하기

- a 태그의 href 속성에 원하는 url을 적어서 사용

- 자세한 내용이 궁금하다면 여기를 클릭하세요
- a태그를 누르면 웹 브라우저의 주소창에 href 속성에 적힌 값을 입력하는 효과가 있다
- 구글가기1(google.com)
- 구글가기2(./google.com)
- 구글가기3(../google.com)
- 구글가기4(/google.com)
- 구글가기5(//google.com)
- 구글가기6(https://google.com)

#웹에서 사용할 수 있는 상대 경로의 종류(,/, ../, /, //)
