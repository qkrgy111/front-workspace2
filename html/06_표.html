# 웹 페이지에 표 출력하기

- table : 표의 전체 범위를 지정하는 태그
- tr : table row, 해당 표에 한 줄을 추가하는 태그
- th : table head, 해당 출에 제목 칸을 추가하는 태그
- td : table data, 해당 줄에 일반 칸을 추가하는 태그
